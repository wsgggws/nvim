This directory contains old or currently unused plugin configs.

Keep these files as references only. Do not import this directory from lazy.nvim.

- dap-python.lua: Old Python DAP configuration. Current Python debugging is handled by LazyVim's Python extra plus dap.lua.
- db.lua: Old database UI/config reference. Current LazyVim extras already load dadbod-related plugins when needed.
- genghis.lua: Old file operation plugin reference.
- goto-preview.lua: Old preview-navigation plugin reference.
- nvim-gomove.lua: Old move-lines/blocks plugin reference.
- printer.lua: Old print/debug helper reference.
- screenkey.lua: Old key display helper reference.
- sniprun.lua: Old code execution plugin reference. Current file execution uses code_runner.nvim.
- telescope.lua: Old fuzzy finder config. Current finder is fzf-lua.
- toggleterm.lua: Old terminal integration config. Current runner uses code_runner.nvim with a bottom terminal.
- vim-easy-align.lua: Old alignment plugin reference.
- vim-swap.lua: Old argument/list swapping plugin reference.
